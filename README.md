# Smart Healthcare Emergency Routing System

## Overview
Smart Healthcare Emergency Routing System is a web-based platform designed to optimize emergency response and hospital routing by managing real-time hospital capacity and ambulance dispatching.

## Features
- Real-time hospital capacity monitoring
- Emergency request handling and ambulance assignment
- Dynamic route optimization using Google Maps API
- Real-time updates and notifications via WebSockets

## Technologies Used
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Real-time communication: WebSockets
- APIs: Google Maps API

## Purpose
This project was developed to demonstrate practical skills in full-stack web development, real-time systems, and API integration, with a focus on scalable and efficient healthcare solutions.
